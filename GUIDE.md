# 🚀 SOHUB AI Vision: Deployment & Automation Guide

এই ফাইলটি আপনার প্রজেক্টের "অটোমেশন গাইড" হিসেবে কাজ করবে। এটি অনুসরণ করলে আপনি প্রোফেশনালদের মতো কোড ম্যানেজ এবং ডেপ্লয় করতে পারবেন।

## 📦 ১. প্রজেক্ট স্ট্রাকচার (Structure)
আপনার প্রজেক্ট দুটি ভাগে বিভক্ত:
- **Frontend (Netlify):** এটি `sohub-vision-spark-main` ফোল্ডারের রিয়্যাক্ট কোড।
- **Backend (Your Server):** এটি `api/` ফোল্ডারের PHP ফাইলগুলো।

---

## 🛠️ স্টেপ-বাই-স্টেপ ডেপ্লয়মেন্ট

### ধাপ ১: গিথাব (GitHub) সেটআপ
১. [GitHub.com](https://github.com) এ আপনি ইতিমধ্যে একটি নতুন **Private Repository** খুলেছেন (নাম: `sohub-ai-vision`)।
২. আপনার কম্পিউটারে টার্মিনাল ওপেন করে এই কমান্ডগুলো রান করুন:
   ```bash
   git init
   git add .
   git commit -m "Initialize SOHUB AI Vision project with dedicated API"
   git branch -M main
   git remote add origin https://github.com/[আপনার-ইউজারনেম]/sohub-ai-vision.git
   git push -u origin main
   ```

### ধাপ ২: নেটলিফাই (Netlify) কানেক্ট করা
১. নেটলিফাই ড্যাশবোর্ড থেকে **"Add new site"** সিলেক্ট করুন।
২. আপনার গিথাব রিপোজিটরি (`ai-vision-spark`) কানেক্ট করুন।
৩. নেটলিফাই অটোমেটিক আপনার সাইটটি বিল্ড করে একটি লিংক দিয়ে দেবে (যেমন: `sohub-ai.netlify.app`)।

### ধাপ ৩: ব্যাকএন্ড (Server) হোস্টিং
যেহেতু নেটলিফাই PHP সাপোর্ট করে না, তাই আপনাকে আপনার ২০২.৫৯.২০৮.১১২ সার্ভারে PHP ফাইলগুলো রাখতে হবে।
১. সার্ভারের `public_html/` এর ভেতরে একটি ফোল্ডার করুন, যেমন: `ai-api/`
২. আপনার লোকাল প্রজেক্টের **`api/`** ফোল্ডারের সব ফাইল (TCPDF সহ) সেখানে আপলোড করুন।
৩. **জরুরি:** সার্ভারে একটি `.env` ফাইল তৈরি করে সেখানে আপনার SMTP (ইমেইল ও পাসওয়ার্ড) সেটিংস দিন।

## 🔐 সিকিউরিটি ও .gitignore
আপনার প্রজেক্টে একটি `.gitignore` ফাইল আছে। এটি নিশ্চিত করে যে আপনার গোপন পাসওয়ার্ড (`.env`) যেন গিথাবে পাবলিক না হয়ে যায়। 
- **কখনো** গিথাবে `.env` পুশ করবেন না। সেটি শুধুমাত্র আপনার পিসিতে এবং আপনার লোকাল/লাইভ সার্ভারে থাকবে।

## 🔄 দৈনন্দিন কাজের ফ্লো (Daily Workflow)
একবার সব সেটআপ হয়ে গেলে, কোড আপডেট করার নিয়ম খুব সহজ:

১. কোডে কোনো পরিবর্তন করুন।
২. টার্মিনালে এই ৩টি কমান্ড দিন:
   ```bash
   git add .
   git commit -m "এখানে লিখুন কি পরিবর্তন করেছেন"
   git push
   ```
৩. **ফল:** নেটলিফাই একা একাই আপনার ওয়েবসাইট আপডেট করে নেবে। আপনাকে আর ম্যানুয়ালি আপলোড করতে হবে না।
